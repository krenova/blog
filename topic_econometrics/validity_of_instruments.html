Consider the first stage equation with the following variables \ m_it is the message sent (dependent variable) z_it is our excluded instrument or message intensity (total messages sent over total existing customers at a given time) X_it are our controls. E.g., RFM marketing metrics First Stage Model and Problem with Message Intensity For simplicity, consider only time fixed effects for the first stage model m_it=δz_it+θX_it+μ_t+ ϵ_it This would yield the corresponding fixed effects model m ̃_it=δz ̃_it+θX ̃_it+ϵ ̃_it where m ̃_it=m_it-m ̅_t, z ̃_it=z_it-z ̅_t and X ̃_it=X_it-X ̅_t. However, since z_it is the proportion of messages sent to all customers at time t, this implies z_it= 1/N ∑_(i=1)^N▒m_it =m ̅_t. Therefore, z ̃_it= z_it-z ̅_t =m ̅_t-m ̅_t=0. The excluded instrument has no variability. First Stage Model and Message Intensity with Random Groups To deal with the issue of no variability, we assign customers into random groups g∈{1,…,|G|}. We write g_i to denote the group customer i is in and write G_i to associate it as the random variable assign i to g_i. Given these groups, as excluded instruments, we calculate the message intensities with respect to these groups as follows, z_it=(∑_(j=1)^N▒〖m_jt 1{G_j=g_i } 〗)/(∑_(j=1)^N▒1{G_j=g_i } )=(∑_(j=1)^N▒〖m_jt 1{G_j=g_i } 〗)/N_(g_i ) =m ̅_(g_i t). Given this definition of message intensity, the resulting average of this instrument is z ̅_t=1/N ∑_(i=1)^N▒z_it =1/N ∑_(i=1)^N▒m ̅_(g_i t) = 1/N ∑_g▒〖N_g m ̅_gt 〗. Hence, the demeaned values of the instruments are z ̃_it= z_it-z ̅_t =m ̅_(g_i t)-1/N ∑_g▒〖N_g m ̅_gt 〗. When the population of customers N studied is large and the group size |G|is small, even though we still expect that E[z_it-z ̅_t ]=E[m ̅_(g_i t)-1/N ∑_g▒〖N_g m ̅_gt 〗]=0 , we know for sure that Var(m ̅_(g_i t)-1/N ∑_g▒〖N_g m ̅_gt 〗)≠0 and that as |G|→∞, Var(m ̅_(g_i t)-1/N ∑_g▒〖N_g m ̅_gt 〗)>Var(m ̅_(〖g'〗_i t)-1/N ∑_g'▒〖N_g' m ̅_g't 〗) for any g^'∈{1,…,|G^' |} such that |G^' |<|G|. Why Randomized Groups make sense To argue the case, let us consider only 2 random groups, g∈{1,2} and consider a particular time t=τ. As an example let the message intensity in τ for the respective groups to be m ̅_1τ=0.8 and m ̅_2τ=0.3. With respect to individuals i and j from group 1 and 2, the probability of a message being sent to them is 0.8 and 0.3 respectively. This illustrates that even though the grouping is random, the magnitude of the message intensity is still proportional to the likelihood of the message being sent to the respective customers. The only problem is that because in our case N, is large we will have m ̅_1t≈m ̅_2t≈m ̅_t for all t and hence the excluded instruments have very little variability and are therefore not very strong instruments. Hence, to circumvent this problem, we can simply increase the number of groups and as long as |G|≪N, we will still have m ̅_gt≈m ̅_t, but what changes is the variability which provides more signal to the model compared to when groupings are small.
